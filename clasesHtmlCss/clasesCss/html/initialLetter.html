Efecto de Letra Capital con CSS

El efecto de letra capital (también conocido como "drop cap") es una técnica tipográfica que se usa para marcar el inicio de un capítulo o una sección de texto, haciendo que la primera letra sea mucho más grande y se mezcle con las primeras líneas del párrafo. Esta propiedad de CSS hace que sea muy fácil lograr este estilo, dándole a tu diseño un toque clásico y profesional.
